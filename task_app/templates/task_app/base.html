Welcome to the Task Management App

This application helps users manage organizations, projects, task statuses, and tasks through a user-friendly interface.

Use the navigation menu to create and manage records. The Secure Access section demonstrates asymmetric key verification, and the Protected Report section is only unlocked after successful verification.
